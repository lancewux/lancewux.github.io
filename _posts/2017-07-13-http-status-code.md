<h1 align="center"> HTTP状态码</h1>

HTTP协议状态码表示的意思主要分为五类 ,大体是 : 

- 1×× 　　保留
- 2×× 　　表示请求成功地接收   
- 3×× 　　为完成请求客户需进一步细化请求   
- 4×× 　　客户错误   
- 5×× 　　服务器错误  

Successful 
-

200 OK

Redirection 
-

301 Moved Permanently

302 Move temporarily

304 Not Modified

307 Temporary Redirect

Client Error 
-

400 Bad Request

401 Unauthorized

403 Forbidden

404 Not Found

407 Proxy Authentication Required

Server Error 
-

500 Internal Server Error 

502 Bad Gateway

503 Service Unavailable

504 Gateway Timeout
