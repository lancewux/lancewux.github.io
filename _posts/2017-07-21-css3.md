<h1 align="center"> CSS3</h1>

CSS3 规范仍在完善中，CSS3 被划分为模块，最重要的模块包括：

- 选择器
- 框模型
- 背景和边框
- 文本效果
- 2D/3D 转换
- 动画
- 多列布局
- 用户界面

边框
-

通过 CSS3，能够创建圆角边框，向矩形添加阴影，使用图片来绘制边框


```html
		<style>
			#css-div {
				width: 200px;
				height: 200px;
				border: 1px solid;
				border-radius:25px; 
				-moz-border-radius:25px; /* Old Firefox */
			}
		</style>
		<div id="css-div"></div>
```
